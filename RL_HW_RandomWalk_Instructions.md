## הסביבה: Random Walk
הסביבה כוללת שרשרת מצבים:
- המצבים הקיצוניים הם **טרמינליים**
- בכל צעד ניתן ללכת **שמאלה** או **ימינה**
- הגעה לטרמינל השמאלי נותנת תגמול `-1`
- הגעה לטרמינל הימני נותנת תגמול `+1`
- בכל שאר המעברים התגמול הוא `0`

ברירת המחדל:
- מספר מצבים: 7
- מצב התחלתי: האמצעי
- מדיניות לדוגמה: בחירה אקראית בין שמאל לימין בהסתברות שווה

# חלק 2 - Model-Free Prediction (40 נקודות)

## שאלה 2.1 - Monte Carlo Prediction
ממשו פונקציה שמעריכה את \( V^\pi(s) \) באמצעות Monte Carlo prediction.

דרישות:
- First-visit Monte Carlo או Every-visit Monte Carlo (יש לציין מה בחרתם)
- הרצת לפחות 1000 אפיזודות
- הצגת הערכים המשוערים לכל מצב
- השוואה לערכים האמיתיים / הצפויים

## שאלה 2.2 - TD(0) Prediction
ממשו פונקציית TD(0) להערכת \( V^\pi(s) \).

דרישות:
- שימוש ב-learning rate \(\alpha\)
- עדכון אונליין לאחר כל מעבר
- השוואה בין התוצאה של TD(0) לבין Monte Carlo

## שאלה 2.3 - השוואה בין האלגוריתמים
בצעו השוואה בין Monte Carlo לבין TD(0) בעזרת:
- טבלה או גרף
- דיון קצר בנושאים:
  - bias / variance
  - מהירות התכנסות
  - תלות באפיזודות מלאות לעומת bootstrapping

# חלק 3 - Value Function Approximation (40 נקודות)

## שאלה 3.1 - Feature Representation
הגדירו ייצוג תכונות \( \phi(s) \) עבור כל מצב.  
ניתן לבחור אחת מהאפשרויות:
- One-hot features
- Polynomial features
- ייצוג אחר באישור המרצה

הסבירו:
- למה בחרתם בייצוג הזה
- מה היתרונות והחסרונות שלו

## שאלה 3.2 - Linear Value Function Approximation
ממשו פונקציה מהצורה:
\[
\hat{V}(s; w) = w^T \phi(s)
\]

והשתמשו ב-TD learning כדי לעדכן את המשקולות:
\[
w \leftarrow w + \alpha \delta_t \phi(s)
\]
כאשר:
\[
\delta_t = r + \gamma \hat{V}(s'; w) - \hat{V}(s; w)
\]

## שאלה 3.3 - ניתוח תוצאות
הציגו:
- גרף של הערכים המנובאים
- השוואה לטבלה הטבלאית (tabular TD)
- דיון:
  - מתי approximation נחוץ?
  - מה מאבדים לעומת טבלה מלאה?
  - כיצד בחירת הפיצ'רים משפיעה על התוצאה?
